Transición de tareas
====================

Las transiciones registran cada movimiento de las tareas entre las columnas.

![Transitions](https://kanboard.net/screenshots/documentation/transitions.png)


Disponible a partir de la vista de tareas, se puede ver la información de:

- Fecha de la acción
- Columna de origen
- La columna Destino
- Ejecutor (usuarios que se mueve la tarea)
- Tiempo de permanencia en la columna de origen

los datos de transición de tareas también se pueden exportar desde la página de configuración del proyecto.

![Transitions Export](https://kanboard.net/screenshots/documentation/transitions-export.png)

Para el rango de tiempo especificado que genera un archivo CSV que se puede utilizar con cualquier software de hoja de cálculo.
