Variables de entorno
=====================

Las variables de entorno pueden resultar utiles cuando se implementa Kanboard en un contenedor (Docker)

| Variable      | Descripción                                                                                                                   |
|---------------|---------------------------------------------------------------------------------------------------------------------------------|
| DATABASE_URL  | `[database type]://[username]:[password]@[host]:[port]/[database name]`, ejemplo: `postgres://foo:foo@myserver:5432/kanboard`   |
| DEBUG         | Enable/Disable debug mode: "true" o "false"                                                                                    |
