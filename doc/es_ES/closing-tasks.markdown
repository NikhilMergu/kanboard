Cerrado de tareas
=============

Cuando una tarea es cerrada, esta se oculta en el tablero

Sin embargo, siempre se puede acceder a la lista de tareas cerradas mediante el uso de la consulta **status:closed** en cualquier formulario de búsqueda o simplemente escoger **Closed tasks** desde el filtro desplegable.

Hay dos maneras diferentes para cerrar una tarea, desde de el drop-down de tareas del tablero:

![Close a task from drop-down menu](https://kanboard.net/screenshots/documentation/menu-close-task.png)

O desde el menú de la barra lateral tarea en la vista detallada de tareas

![Close task](https://kanboard.net/screenshots/documentation/closing-tasks.png)

Nota: Cuando se cierra una tarea, todos los sub-tareas no completadas se cambiará al estado "Listo".
