Tipos de usuario
================

En Kanboard hay dos tipos de usuarios:

| Type               | Description                                                           |
|--------------------|-----------------------------------------------------------------------|
| Usuarios Locales   | Del usuario que almacena su contraseña en la base de datos de Kanboard|
| Usuarios Remotos   | Las credenciales de usuario se gestionan mediante otro sistema (Ejemplo: LDAP server) |

Ejemplos de usuarios remotos:

- Usuario LDAP
- Los usuarios autenticados por un reverse-proxy
