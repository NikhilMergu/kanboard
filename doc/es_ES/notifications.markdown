Notificaciones
=============

Kanboard es capaz de enviar notificaciones a través de varios canales:

- Email
- Web (Lista de mensajes no leídos)

plugins externos le permiten enviar notificaciones a Slack, HipChat, Jabber o cualquier sistema de chat.

Configuración
-------------

Cada usuario debe activar las notificaciones en su perfil: **Perfil del usuario> Notificaciones**. Está desactivado por defecto.

Para recibir notificaciones por correo electrónico se necesita una dirección válida de correo electrónico en su perfil y la aplicación debe estar configurado para enviar correos electrónicos.

![Notifications](https://kanboard.net/screenshots/documentation/notifications.png)

Se puede elegir el tipo de notificación favorita:

- Los correos electrónicos
- Web (ver más abajo)

Para cada proyecto que usted es miembro, puede optar por recibir notificaciones para:

- Todas las tareas
- Sólo para las tareas que tiene asignadas
- Sólo para tareas creadas por usted
- Sólo para tareas creadas por usted y que tiene asignadas

También puede seleccionar sólo algunos proyectos, por defecto es todos los proyectos en los que es miembro.


Notificaciones Web
-----------------

Las notificaciones Web están disponibles desde el panel de control o desde el icono en la parte superior:

![Web Notifications Icon](https://kanboard.net/screenshots/documentation/web-notifications-icon.png)

Las notificaciones se muestran en una lista, por lo que puede marcar la notificación individual o todas.

![Web Notifications](https://kanboard.net/screenshots/documentation/web-notifications.png)

De esta manera usted todavía puede recibir una notificación sin tener que recibir mensajes de correo electrónico.
