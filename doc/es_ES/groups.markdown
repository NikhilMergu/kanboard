Administración de grupos
========================

En Kanboard, cada usuario puede ser miembro de  uno o mas grupos.
Un grupo es como un equipo o una organización.

Algunos administradres pueden crear nuevos grupos y asignarlos a usuarios.

Los grupos pueden ser administrados desde **Administración de usuarios > Ver todos los grupos**.
A partir de ahí, se pueden crear grupos y asignar usuarios.

![Group Management](screenshots/groups-management.png)

Cada jefe de proyecto puede autorizar el acceso a un conjunto de grupos desde [pagina de permisos de proyecto](project-permissions.markdown).

El ID externo se utiliza principalmente para los proveedores de grupos externos.
Kanboard proporciona un proveedor de grupo LDAP para[sincronizar automaticamente grupos desde el servidor LDAP](ldap-group-sync.markdown).
