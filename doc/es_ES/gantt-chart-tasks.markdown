Graficas de Gantt para tareas
==============================

El objetivo de este diagrama de Gantt es mostrar una visión general basada en el tiempo de las tareas de un proyecto determinado.

- El diagrama de Gantt está disponible en el "selector de visualización"
- Sólo los administradores del proyecto pueden acceder a esta sección

![Gantt Chart](https://kanboard.net/screenshots/documentation/gantt-chart-project.png)

- La **fecha de inicio**  y la **fecha de vencimiento** de tareas se utilizan para dibujar el gráfico
- Las tareas se pueden clasificar según el tamaño y se mueven horizontalmente con el ratón
- No hay **drag and drop** en vertical
- La barra es del mismo color que la tarea
- Cada barra muestra un estado de progresión en porcentaje, este porcentaje se calcula utilizando la posición de la columna en el tablero
- Para encajar con el modelo Kanban, las tareas pueden ser ordenados por la tabla de posiciones o por la fecha de inicio
- Las nuevas tareas creadas a partir de este punto de vista se mostrarán en el tablero en la posición 1 en la primera columna
- Las tareas se muestran en negro cuando no hay inicio o de caducidad definida
