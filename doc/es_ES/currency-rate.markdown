Currency Rate **Presupuestos**
==============================

Dado que cada usuario puede tener una tarifa honoraria predefinida en diferentes monedas.
Si usted tiene que manejar múltiples monedas, se define aquí la tasa de acuerdo a la moneda de referencia.

Esta función se utiliza para el cálculo del presupuesto del proyecto.
![Currency Rate](https://kanboard.net/screenshots/documentation/currency-rate.png)

Estos ajustes de  moneda se encuentran en **Settings > Currency rates**.
