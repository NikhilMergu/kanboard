configuración de links
=======================

Las relaciones de tareas se pueden cambiar desde la configuración de la aplicación (**Configuración > Configuración del enlace**)

![Link Labels](https://kanboard.net/screenshots/documentation/link-labels.png)

Cada etiqueta puede tener una etiqueta opuesta definida.
Si no hay opuesto, la etiqueta se considera Bidireccional.

![Link Label Creation](https://kanboard.net/screenshots/documentation/link-label-creation.png)
