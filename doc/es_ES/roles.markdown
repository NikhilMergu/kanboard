Roles de usuario
================

Roles de aplicación
-----------------

Cada usuario de Kanboard tiene uno de estos roles:


| Role          | Descripción                                                |
|---------------|-----------------------------------------------------------------|
| Administrator | El acceso a todo                                           |
| Manager       | Puede crear proyectos de equipo, pero no puede cambiar la configuración de aplicaciones |
| User          | Sólo se puede crear proyectos privados                               |


Roles de proyecto
-----------------

Cada proyecto de equipo individual puede asignar una función diferente a cada usuario y grupo:

| Role            | Description                                                          |
|-----------------|----------------------------------------------------------------------|
| Project Manager | Puede cambiar la configuración del proyecto, acceder al gráfico de Gantt y los informes  |
| Miembro del Proyecto  | Puede crear tareas y usar el tablero                                  |
| Project Viewer  | Acceso de sólo lectura al tablero y tareas                       |
